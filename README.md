# Central Asia Ethnicities Map

Interactive map of different data collected from government statistic publications across Central Asia using census and boundary data. This is an ongoing project I made just for personal interest because I find the area really fascinating. If there is any data you think would be interesting to add to the map let me know!

## Stack
- Python for data processing
- React + Vite + Tailwind for UI
- Leaflet for map rendering

## Folders
- `data/raw` source files
- `data/processed` final outputs for the front end
- `scripts` ETL pipeline
- `web` front-end app

## Processing Data 
- clean 
- inspect 
- prepare/reshape
