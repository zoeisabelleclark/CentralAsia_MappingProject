# Central Asia Atlas

Central Asia Atlas is an interactive web map for exploring region-level demographic data across Kazakhstan, Kyrgyzstan, Uzbekistan, Tajikistan, and Turkmenistan. This is an ongoing project I made just for personal interest because I find the area really fascinating. If there is any data you think would be interesting to add to the map let me know!

Website is live [here](https://centralasiaatlas.com/)

## Topics
central-asia, geospatial, react, leaflet, python, demographics, data-visualization

## Stack
- Python for data processing
- React + Vite + Tailwind for UI
- Leaflet for map rendering

## Folders
- `data/raw` source files
- `data/processed` final outputs for the front end
- `scripts` ETL pipeline
- `web` front-end app
