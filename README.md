# Central Asia Ethnicities Map

Interactive map of ethnicities spoken across Central Asia using census and boundary data. ONGOING UNFINISHED PROJECT

## Stack
- Python for data processing
- React + Vite + Tailwind for UI
- Leaflet for map rendering

## Folders
- `data/raw` source files
- `data/processed` final outputs for the front end
- `scripts` ETL pipeline
- `web` front-end app

## First run
1. Create and activate virtualenv
2. Install Python requirements
3. Build processed GeoJSON and JSON
4. Start front end
